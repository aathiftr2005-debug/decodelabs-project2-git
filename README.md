# Project 2: Version Control with Git

## Overview
Managed project code using Git and GitHub with proper 
version control workflow.

## Git Workflow Used
Working Directory → Staging Area → Repository

## Commands Practiced
| Command | Purpose |
|---------|---------|
| git init | Initialize repository |
| git add | Stage changes |
| git commit | Save snapshot |
| git push | Sync to GitHub |
| git status | Check current state |
| git log | View commit history |
| git branch | Create branches |
| git merge | Merge branches |

## Tasks Completed
- Created GitHub repository
- Made 8+ meaningful commits
- Followed trunk-based development
- Written professional commit messages
